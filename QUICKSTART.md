# 快速開始指南

## 第一步：安裝依賴

```bash
npm install
```

## 第二步：啟動開發伺服器

```bash
npm run dev
```

瀏覽器會自動打開 http://localhost:3000

## 第三步：開始使用

1. **新增故障報告**：點擊右上角的「+ 新增故障報告」按鈕
2. **查看統計**：在頁面頂部查看故障統計數據
3. **篩選故障**：使用狀態和優先級篩選器查看特定故障
4. **管理故障**：點擊故障卡片查看詳情並更新狀態

## 資料庫

- 資料庫檔案會自動創建在 `data/faults.db`
- 首次運行時會自動初始化資料表結構
- 無需額外配置

## 故障狀態說明

- **待處理 (pending)**：新提交的故障報告
- **處理中 (in_progress)**：正在處理的故障
- **已解決 (resolved)**：故障已修復
- **已關閉 (closed)**：故障已關閉

## 優先級說明

- **低 (low)**：不緊急，可延後處理
- **中 (medium)**：一般優先級
- **高 (high)**：需要優先處理
- **緊急 (urgent)**：需要立即處理

## 常見問題

### Q: 資料庫檔案在哪裡？
A: 在專案根目錄的 `data/faults.db` 檔案

### Q: 如何重置資料庫？
A: 刪除 `data/faults.db` 檔案，重新啟動應用程式即可

### Q: 如何備份資料？
A: 複製 `data/faults.db` 檔案即可
